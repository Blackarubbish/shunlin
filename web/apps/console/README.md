# 博客管理后台

一个基于 React + TypeScript + Ant Design + Tailwind CSS v4 构建的现代化博客管理后台系统。

## 功能特性

### ✨ 核心功能

- **🏠 仪表盘** - 博客数据概览和快速操作
- **📝 文章管理** - 文章的创建、编辑、发布和管理
- **🏷️ 分类管理** - 博客分类的增删改查
- **📁 媒体库** - 图片、视频等媒体文件的上传和管理
- **👥 用户管理** - 系统用户和权限管理

### 🎨 设计特点

- **现代极简风格** - 简洁优雅的界面设计
- **响应式布局** - 完美适配各种设备尺寸
- **统一设计系统** - 基于 Tailwind CSS v4 的设计规范
- **优秀用户体验** - 符合现代用户操作习惯

### 🛠️ 技术栈

- **React 19** - 最新的 React 框架
- **TypeScript** - 类型安全的 JavaScript
- **Ant Design 5.x** - 企业级 UI 组件库
- **Tailwind CSS v4** - 原子化 CSS 框架
- **React Router v7** - 客户端路由管理
- **Vite** - 快速的构建工具

## 页面结构

```
/                   → 重定向到 /dashboard
├── /dashboard      → 仪表盘首页
├── /posts          → 文章列表
│   ├── /create     → 创建文章
│   └── /edit/:id   → 编辑文章
├── /categories     → 分类管理
├── /media          → 媒体库
└── /users          → 用户管理
```

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
pnpm run dev
```

### 构建生产版本

```bash
pnpm run build
```

### 预览生产版本

```bash
pnpm run preview
```

## 项目结构

```
src/
├── components/          # 公共组件
│   └── layout/         # 布局组件
├── pages/              # 页面组件
│   ├── dashboard/      # 仪表盘
│   ├── posts/          # 文章管理
│   ├── categories/     # 分类管理
│   ├── media/          # 媒体库
│   └── users/          # 用户管理
├── router/             # 路由配置
├── types/              # TypeScript 类型定义
├── utils/              # 工具函数和模拟数据

├── App.css             # 全局样式
└── main.tsx            # 应用入口
```

## 主要页面功能

### 📊 仪表盘
- 博客数据统计（文章数、分类数、用户数等）
- 最近文章列表
- 热门文章排行
- 快速操作入口

### 📝 文章管理
- 文章列表展示和筛选
- 富文本编辑器
- 文章状态管理（草稿/已发布/已归档）
- 分类和标签管理
- SEO 设置
- 媒体文件插入

### 🏷️ 分类管理
- 分类的创建、编辑和删除
- 分类统计信息
- 分类别名（slug）管理

### 📁 媒体库
- 拖拽上传文件
- 图片预览
- 文件类型筛选
- 批量操作
- 文件链接复制

### 👥 用户管理
- 用户信息管理
- 角色权限设置（管理员/编辑）
- 账户状态控制

## 样式系统

项目使用 Tailwind CSS v4 的新特性：

- `@theme` 规则定义设计令牌
- `@layer` 规则组织样式层级
- CSS 变量支持
- 与 Ant Design 的完美集成

## 数据模拟

项目包含完整的模拟数据：

- 文章数据（已发布、草稿）
- 分类数据
- 媒体文件数据
- 统计数据
